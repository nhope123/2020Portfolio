Introduction

What is the shell? The shell is basically a program that takes your commands from the keyboard and sends them to the operating system to perform. If you’ve ever used a GUI, you’ve probably seen programs such as “Terminal” or “Console” these are just programs that launch a shell for you. Throughout this entire course we will be learning about the wonders of the shell.

we will use the shell program bash (Bourne Again shell), almost all Linux distributions will default to the bash shell. There are other shells available such as ksh, zsh, tsch, but we won’t get into any of those.

Depending on the distribution your shell prompt might change, but for the most part it should adhere to the following format, where the $ at the end of the prompt:

username@hostname:current_directory
pete@icebox:/home/pete $

ls

The ls (List Directories) command will list directories and files in the current directory by default, however you can specify which path you want to list the directories of.

Options:

-a
List all files, including hidden .files.
-l
Use a long listing format.
-t
Sort by modification time, newest first.
-S
Sort by file size, largest first

Show hidden files in current directory

.    Documents  Music
..  .snake.txt  index.html

List with long format and show hidden files

$ ls -lta
total 6
drwxr-xr-x 15 pete penguingroup 4096 Jun 28 07:56 .
drwxr-xr-x 45 pete penguingroup 4096 Jun 23 21:52 ..
drwxr-x--- 2 pete penguingroup 4096 Oct 19 10:46 Documents

Cut

The cut command extracts sections from each line of files

Options:

-c
Select only these characters
-d
Use delimiter instead of TAB for field delimiter
-f
Select only these fields
-s
Do not print lines not containing delimiter
--complement
Complement the set of selected bytes, characters or fields
--output-delimiter=STRING
use STRING as the output delimiter

Syntax

$ cut option filename

Select column of characters using range

$ cut -c1-6 text.txt

Select a specific field from files

$ cut -f5 text.txt

Select Fields Only When a Line Contains the Delimiter

$ cut -d':' -f7 text.txt

Select only Fields Specified in Fields

$ cut -d':' -s -f7 text.txt

Select All Fields not Specified

$ cut -d':' --complement -f7 text.txt

Change Output Delimiter for Display

$ cut -d':' -f1,6,7 --output-delimiter='#' text.txt

Tar

The tar command is an archiving utility which archive many files into a neat single file. This single file carries a .tar file extension. The bzip or gzip compression utility, combine with the tar utility, reduces the file size.

Options:

c
- Create a archive file.
x
- Extract a archive file.
v
- Show the progress of archive file.
f
- Filename of archive file.
t
- Viewing content of archive file.
j
- Filter archive through bzip2.
z
- Filter archive through gzip.
r
- Append or update files or directories to existing archive file.
w
- Verify a archive file.

Archiving Syntax

tar [option] tarfile.tar [files_to_archive]
$ tar cvf mytarfile.tar mycoolfile1 mycoolfile2

To add compression with gzip

$ tar czvf mytarfile.tar.gz mycoolfile1 mycoolfile2

To create an archive and exclude particular files

$ tar cvf mytarfile.tar --exclude=\*.{jpg,gif,png} /location/of/files/

Extracting Syntax

tar [option] archive_file.tar
$ tar xvf mytarfile.tar

To uncompress with gzip

$ tar xzvf mytarfile.tar.gz

Show content of compressed archive

$ tar ztvf archive_file.tar.gz

Find

The Linux find command is a very useful and handy command to search for files from the command line. It can be used to find files based on various search criterias like permissions, user ownership, modification date/time, size etc.

Options:

-d
list directories themselves, not their contents
-name
Searches for files by their name
-iname
Ignore the case when searching for file names

Syntax

find location comparison-criteria search-term

Search for only directories

$ find ./test -type d -name "abc*"
./test/abc

Search specific directory or path

$ find ./test -name "abc.txt"
./test/abc.txt

Ignore the case

$ find ./test -iname "*.Php"
./test/subdir/how.php
./test/cool.php

Grep

The grep command is a text processing utility. It searches a file for characters that match a certain pattern and print the matching lines.

Options:

-e
Match patterns
-i
Ignore case distinction
-l
Suppress normal output and print the name of the first matching input file.
-n
Show line numbers as well as the matching lines
-A num
Print num of lines after matching lines.
-B num
Print num of lines before matching lines.

Syntax

grep -option PATTERN files
$ grep -i somepattern somefile

Find Fan or Gan in all files in the current dir

$ grep '[FG]an' *

Show filenames containing case-insensitive pattern

$ grep -il string *.html

Show matching line numbers

$ grep -n we address.txt

Print lines before and after grep match

$ grep -B5 -A5 "st" address.txt

Search multiple patterns

$ grep -e 'car|van|suv' driver.txt

Grep in a pipeline

$ ps aux | grep httpd
